# 自定义相机页面使用说明

## 功能特点

1. **全屏相机预览** - 使用 camera 组件实现原生相机体验
2. **顶部导航栏** - 返回按钮、标题、闪光灯切换、前后镜头切换
3. **帮助按钮** - 显示当前模式的拍摄提示
4. **中间提示** - 根据当前模式显示不同的拍摄提示
5. **底部模式切换** - 5 种拍摄模式（拍药盒、拍报告、拍饮食、拍皮肤、扫药码）
6. **底部控制区** - 相册选择、拍照按钮、镜头切换

## 使用方法

### 1. 在 app.json 中注册页面

```json
{
  "pages": [
    "pages/camera/camera"
  ]
}
```

### 2. 跳转到相机页面

```javascript
// 方式1：普通跳转
wx.navigateTo({
  url: '/pages/camera/camera'
});

// 方式2：指定初始模式
wx.navigateTo({
  url: '/pages/camera/camera?mode=拍报告'
});
```

### 3. 图标资源

需要在 `pages/camera/images/` 目录下放置以下图标：

| 文件名 | 用途 | 建议尺寸 |
|--------|------|----------|
| back.png | 返回按钮 | 40x40 |
| flash-off.png | 闪光灯关闭 | 32x32 |
| flash-on.png | 闪光灯开启 | 32x32 |
| flash-auto.png | 闪光灯自动 | 32x32 |
| switch.png | 切换镜头 | 32x32 |
| album.png | 相册 | 56x56 |
| flip.png | 翻转镜头 | 56x56 |

如果没有图标文件，可以先用文字代替或从 iconfont 下载。

## 注意事项

1. **权限配置** - 需要在小程序后台「设置」→「接口设置」中申请相机权限
2. **真机调试** - camera 组件在模拟器中可能无法正常显示，建议在真机上测试
3. **安全区域适配** - 代码已适配刘海屏和底部安全区域

## 扩展建议

1. 可以接入 OCR 识别 API 实现拍药盒自动识别药品信息
2. 可以接入条码识别 API 实现扫药码功能
3. 拍照后可以添加图片编辑功能（裁剪、旋转等）
